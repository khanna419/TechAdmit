# TechAdmit

TechAdmit is a web-based Engineering Admission Portal developed using PHP, MySQL, HTML, CSS, and JavaScript.

## Features
- Student Registration and Login
- Course Browsing
- Admission Application Submission
- Application Tracking
- Session Management
- Secure Form Validation
- MySQL Database Integration

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Project Overview
TechAdmit simplifies the engineering admission process by allowing students to register, explore courses, submit applications, and track admission status through an interactive web portal.
